# Ad Campaign Agent — Fields Schema

The agent validates user input against this schema before generating the creative brief.
If required fields are missing, the agent enters **Loop 1** to ask the user.

---

## Required Fields

| # | Field | Type | Description | Example |
|---|-------|------|-------------|---------|
| 1 | **Product / Service Name** | string | What is being advertised | "Nike Air Max 2026" |
| 2 | **Target Audience** | string | Demographics, interests, psychographics | "Gen Z sneaker enthusiasts, 18-25, urban" |
| 3 | **Campaign Goal** | enum | Primary objective | `awareness` · `consideration` · `conversion` · `launch` |
| 4 | **Key Message / CTA** | string | Core takeaway + call to action | "Step into the future — Shop now at nike.com" |
| 5 | **Brand Tone** | string | Emotional tone and personality | "Bold, energetic, aspirational" |
| 6 | **Style Reference** | preset / upload | Visual direction — selected in sidebar | "Futuristic" or custom upload |

## Optional Fields

| # | Field | Type | Default | Description |
|---|-------|------|---------|-------------|
| 7 | **Brand Colors** | hex[] | Inferred from style | Primary and accent colors |
| 8 | **Logo** | image file | None | Brand logo to overlay on the image |
| 9 | **Competitor References** | string | None | Ads or styles to differentiate from |
| 10 | **Existing Tagline** | string | None | Established brand tagline |
| 11 | **Do-Not-Include** | string | None | Elements to avoid (competitors, sensitive topics) |

---
